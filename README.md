# 项目简介

---

基于仓储物流识别的人工智能挑战项目，本仓库仅仅有训练代码及相关文档，不包含项目本身的所有程序。



# 项目结构

---

项目分为图像分类以及图像检测两个板块

```python
project
│  baidu_api.py 									#封装好的百度API实例
│  getdata.py									    #数据清洗（伪）
│  img_deticion.py									#视频流的图像分类
│  img_classfication.py								#视频流的图像检测
│  README.md									    #说明文档
│  基于百度EasyDL的仓储物流识别技术文档V1.0-李星辰.docx  #技术文档
```
